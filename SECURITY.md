# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it as soon as possible.

You can report vulnerabilities by:
- Opening an issue in this repository
- Sending an email to **rafaelomarques@gmail.com**

When reporting, please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Any relevant context or impact

Reports will be reviewed and addressed as time permits. If the vulnerability is confirmed, a fix will be released as soon as possible.
